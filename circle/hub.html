Comandos del generador
Python 3.8+ · Sin dependencias externas · Corre desde la misma carpeta que el template y el JSON.
1Ver todos los embajadores registrados
python3 generar_smart_pass.py --lista
2Generar todos los passes activos
python3 generar_smart_pass.py # Solo embajadores con estado "activo" # Output: ./output_smart_passes/Smart_Pass_[slug]_[PASS-ID].html
3Generar un embajador específico
python3 generar_smart_pass.py anais-hidalgo python3 generar_smart_pass.py carlos-mendez python3 generar_smart_pass.py edwin-rodriguez # Acepta slug completo o nombre parcial
4Ver fotos pendientes por embajador
python3 generar_smart_pass.py --pendientes # Lista exactamente qué archivos faltan # con el nombre exacto que deben tener al subirse
5Generar absolutamente todos (activos + pendientes)
python3 generar_smart_pass.py --todos
6Deploy manual de un Smart Pass
# 1. Renombrar el HTML generado mv Smart_Pass_anais-hidalgo_PASS-SLS-2026-ANA-H.html index.html # 2. Subir por FTP/SFTP a la carpeta del embajador soullensstudios.live/embajador/anais-hidalgo/ # 3. Subir fotos (nomenclatura exacta) anais-hero.jpg ← 1080×1920px anais-perfil.jpg ← 400×400px og.jpg ← 1200×630px (preview WhatsApp) anais-01.jpg ← 400×576px (hasta 09.jpg)
7Verificar Open Graph antes de compartir
# Validar preview en WhatsApp / redes https://developers.facebook.com/tools/debug/ # URL a probar: https://soullensstudios.live/embajador/anais-hidalgo/
8Flujo operativo completo — nuevo embajador
# 1. Embajador llena el formulario circle.soullensstudios.live/unete/ (SC) circle.soullensstudios.live/nucleo/ (NS) # 2. Embajador descarga y envía su JSON al equipo SLS # 3. Equipo SLS agrega el JSON a embajadores.json # 4. Generar Smart Pass python3 generar_smart_pass.py [slug] # 5. Subir HTML + fotos al servidor # 6. URL activa: soullensstudios.live/embajador/[slug]/ # 7. Embajador comparte su Smart Pass
