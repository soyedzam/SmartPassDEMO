Flujo de deploy por embajador
1
Generar el Smart Pass desde el dashboard
Ir a la vista "Embajadores" → seleccionar embajador → "Generar Smart Pass" → descargar el HTML.
2
Renombrar el archivo generado
mv Smart_Pass_slug_PASS-SLS-2026-XXX-X.html index.html
3
Subir por FTP/SFTP a la carpeta del embajador
# Estructura en el servidor: soullensstudios.live/embajador/[slug]/ index.html [nombre]-hero.jpg [nombre]-perfil.jpg og.jpg [nombre]-01.jpg … [nombre]-09.jpg
4
Verificar Open Graph (preview en WhatsApp)
Usar el debugger de Facebook para validar que la imagen og.jpg aparece al compartir el link.
↗ Facebook Debugger ↗ Twitter Card Validator
